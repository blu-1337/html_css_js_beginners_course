Instructions

It's time to practice working with the input event!  In the index.html file, you'll find an <h1> and an <input type="text"> element.  Add the app.js script to the intex.html before the ending of the body element and afterwards do not change anything else in index.html!  In app.js, write code that meets these requirements:

  • The h1 should start with the text "Enter Your Username" (I've done that for you, already in the markup)

  • Whenever an input event is fired on the <input> element, update the <h1> so that it displays "Welcome, " plus the current value from the text input.  Take a look at the gif below to see how it should work.

  • If the <input> goes back to being empty, update the <h1> so that it once again says "Enter Your Username"

The result should be as in the following image. Here we can see that the page starts with enter your username, then as we begin to type, it should change to "Welcome, " and then whatever the user is typing. The picture represents the website at different time points.

The h1 is updating as I type, rather than whenever I leave the input.  Be sure to listen for an input event, rather than a change event.
